MotionCurve

Visualize and experiment with CSS easing functions. Animate a box with built-in or custom cubic-bezier curves, control the progress, and preview easing on a curve editor.

  • Easing Selector: Choose from common easing functions or create your own cubic-bezier curve.
  • Animation Controls: Play, pause, replay animations and toggle auto replay.
  • Progress Slider: Scrub through the animation timeline manually.
  • Bezier Editor: Visually edit cubic-bezier curves with draggable points and see the live easing preview.
  • Copy Code: Copy the cubic-bezier code to use in your CSS.
  • Dark Mode: Toggle dark mode for comfortable viewing.
